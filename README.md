# 🚀 Terraform AWS Tasks – 100 Days of DevOps

This repository contains hands-on Terraform solutions for AWS services, created as part of my **100 Days of DevOps journey**.

---

## 📌 Overview

The tasks focus on building and managing AWS infrastructure using **Infrastructure as Code (IaC)** with Terraform.

Covered AWS services:
- ✅ VPC (IPv4 & IPv6)
- ✅ S3 Buckets (Public & Private)
- ✅ DynamoDB
- ✅ Kinesis Streams
- ✅ IAM (Users & Policy Attachments)
- ✅ EC2 & Elastic IP Association
- ✅ Key Pair Generation
